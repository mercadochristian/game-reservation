Full-stack volleyball game reservation system with three roles: Admin, Facilitator, Player.

Stack: Next.js 15 (App Router), React 19, TypeScript 5 (strict mode enabled), Tailwind CSS v4, Supabase (PostgreSQL + Auth), Framer Motion v12, React Hook Form + Zod v4, Sonner toasts, Lucide React icons.

**Why:** Personal project being built feature by feature.

**How to apply:** Use this as context when designing new features — role-scoped pages under `src/app/{role}/`, Zod v4 API (`.issues` not `.errors`), and always pull shared constants/animations from central lib files.
